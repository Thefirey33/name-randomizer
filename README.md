# Name Randomizer

A small little program I made in C++, CMake and Qt, for randomly picking names out of a list.

**Reminder:** This program is entirelty in Turkish. As it was made due to a request from a turkish teacher.

## Requirements

- C++ 11 
- Qt6
- CMake 3.10

## Building & Running

Confirm that you are in the directory of the root of the project, and type:

```bash
cmake --build .
```

After the application has been built, go to the `build/` folder, and click on the executable.
